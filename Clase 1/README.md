# Sistema de Información Personal – Enunciado de Ejemplo en Python

## Descripción del ejemplo

Este ejemplo consiste en implementar un pequeño sistema interactivo que solicite datos personales al usuario, valide la información ingresada y realice varios cálculos básicos.  
El programa utiliza conceptos fundamentales de Python como **variables, casteos, condicionales, funciones, operadores aritméticos, bucles y validaciones de entrada**.

El sistema debe:

- Solicitar el nombre y la edad del usuario.
- Validar que la edad sea un entero positivo.
- Calcular el año estimado de nacimiento.
- Clasificar al usuario según su rango de edad.
- Solicitar si tiene identificación válida (DPI) y verificar acceso.
- Realizar una cuenta regresiva usando un ciclo `while`.
- Mostrar una pequeña repetición usando un ciclo `for`.
- Determinar si la edad ingresada es par o impar.

Todo esto utilizando **buenas prácticas de programación**, funciones claras y manejo apropiado de cadenas.

## Palabras clave y significado

| Palabra       | Significado                                                   |
|---------------|---------------------------------------------------------------|
| `variable`    | Representa un valor almacenado en memoria                    |
| `input()`     | Permite leer datos ingresados por el usuario                 |
| `int()`       | Convierte una cadena numérica en entero                      |
| `str()`       | Convierte un valor en cadena                                 |
| `if / elif / else` | Estructura para tomar decisiones                       |
| `and / or`    | Operadores lógicos                                           |
| `def`         | Define una función                                           |
| `return`      | Retorna un valor desde una función                           |
| `while`       | Repite instrucciones mientras la condición sea verdadera     |
| `for`         | Itera una cantidad definida de veces                         |
| `range()`     | Generador numérico para ciclos                               |
| `docstring`   | Comentario especial que documenta una función                |
| `validación`  | Proceso de comprobar que el dato ingresado es válido         |

## Requerimientos del Sistema

El programa debe cumplir con lo siguiente:

1. Solicitar el nombre del usuario.  
2. Solicitar la edad y validarla para que sea un entero positivo.  
3. Calcular el año de nacimiento usando una función.  
4. Clasificar al usuario como:  
   - Menor de edad  
   - Adulto  
   - Adulto mayor  
5. Solicitar si tiene DPI (`s/n`) y validar la entrada.  
6. Determinar si el usuario obtiene acceso según su edad y su respuesta.  
7. Realizar:  
   - Una cuenta regresiva con un ciclo `while`.  
   - Una secuencia de impresiones con un ciclo `for`.  
8. Verificar si la edad es par o impar.  
9. Utilizar funciones, comentarios y separación lógica del código.
