1. ¿Qué es HTML?

HTML estructura el contenido de una página web. Ejemplos de etiquetas comunes:

  • <h1> - <h6>: títulos
  • <p>: párrafos
  • <div>: contenedor genérico
  • <span>: contenedor en línea
  • <a>: enlaces
  • <img>: imágenes
